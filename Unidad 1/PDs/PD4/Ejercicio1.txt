Dado el siguiente código fuente:
public class IdentifyMyParts {
	public static int x = 7;
	public int y = 3;
}
a) ¿Cuáles son las variables de clase?

las variables de clase son identificadas con static, en este caso x

b) ¿Cuáles son las variables de instancia?

las variables de instancia no son denotadas con nada en particular, se instancian en el entorno que se precisen. En este caso y.

¿Cuál es la salida que produce el siguiente código?
IdentifyMyParts a = new IdentifyMyParts();
IdentifyMyParts b = new IdentifyMyParts();
a.y = 5;
b.y = 6;
a.x = 1;
b.x = 2;
System.out.println("a.y = " + a.y);
System.out.println("b.y = " + b.y);
System.out.println("a.x = " + a.x);
System.out.println("b.x = " + b.x);
System.out.println("IdentifyMyParts.x = " + IdentifyMyParts.x);

EL resultado seria:

a.y = 5
b.y = 6
a.x = 2
b.x = 2
IdentifyMyParts.x = 2

Esto ocurre ya que x es variable de clase por lo que al modificarlo se modifica en todos los lbjetos, y el ultimo cambio es a 2 por lo que todos los x termian siendo 2.
